# SIR 지수 가중치 및 감성 처리 스펙

## 1. 개요

SIR(Social Investment Risk) 지수는 다양한 채널에서 수집한 감성 데이터를 **채널 신뢰도 가중치**와 **채널별 감성 점수**를 조합하여 하나의 수치로 산출한다.

---

## 2. 채널별 가중치 (Channel Weight)

| 채널                 | 원점수 | 정규화 가중치 (Wc) |
| -------------------- | ------ | ------------------ |
| 기업분석보고서       | 5      | 0.38               |
| 뉴스                 | 4      | 0.31               |
| SNS (블로그/유튜브)  | 2      | 0.15               |
| 커뮤니티 (종토방 등) | 2      | 0.15               |

> **정규화 계산**: Wc = 원점수 / 합계(13)
> 특정 채널에 데이터가 0건이면 해당 채널 제외 후 나머지 가중치를 재정규화

---

## 3. 채널별 감성 점수 (Channel-Specific Sentiment Score)

### 3.1 채널별 감성 가중치 매트릭스

각 채널마다 감성의 신뢰도와 시장 영향력이 다르므로, 채널별로 감성 점수를 다르게 적용한다.

| 감성       | 보고서 | 뉴스  | SNS   | 커뮤니티 | 근거                                           |
| ---------- | ------ | ----- | ----- | -------- | ---------------------------------------------- |
| positive   | +1.0   | +1.0  | +0.7  | +0.5     | 저신뢰 채널의 긍정은 펌핑/과장 가능성 할인     |
| neutral    | 0      | 0     | 0     | 0        | 모든 채널에서 중립은 0 (분모에만 포함)          |
| negative   | -1.3   | -1.5  | -0.8  | -0.7     | 고신뢰 채널 부정은 증폭, 저신뢰 채널은 할인    |
| critical   | -2.0   | -2.0  | -1.5  | -1.5     | 허위사실/비방은 채널 불문 강한 패널티           |

### 3.2 설계 근거

**고신뢰 채널 (보고서, 뉴스)**
- 긍정: 검증된 호재이므로 full weight (+1.0)
- 부정: 검증된 악재는 시장 충격이 긍정보다 큼 (negativity bias → -1.3, -1.5)
- 뉴스의 부정(-1.5)이 보고서(-1.3)보다 강한 이유: 뉴스는 대중에게 즉시 노출되어 패닉 유발

**저신뢰 채널 (SNS, 커뮤니티)**
- 긍정: 펌핑/과장이 빈번하므로 할인 (SNS +0.7, 커뮤니티 +0.5)
- 부정: FUD도 있지만 조기 경보 가능성도 있으므로 적당히 반영 (-0.8, -0.7)
- 커뮤니티 긍정(+0.5)이 가장 낮은 이유: "무조건 오른다" 류의 노이즈가 가장 많음

### 3.3 중립(neutral) 처리

**중립은 무시하지 않는다. 점수 0으로 반영한다.**

- 중립 건수가 많다 = 시장이 관심은 있으나 방향성 판단이 안 되는 상태
- 중립이 분모(전체 건수)에 포함되면 지수가 0(중립) 쪽으로 희석
- 이 희석 효과가 현실을 정확히 반영 (관심은 있으나 방향 모름 = 리스크 중간)

## 4. SIR 지수 계산 공식

### 4.1 공식

```
SIR = Σ ( Wc × (채널별 가중감성합 / 채널별 건수) )
```

여기서 '가중감성합'은 채널별 감성 점수 매트릭스를 적용한 합산:

```
채널별 가중감성합 = Σ (각 기사의 채널별_감성점수)
```

### 4.2 계산 예시

```
뉴스 채널: 긍정 5건, 중립 8건, 부정 2건

가중감성합 = (5 × 1.0) + (8 × 0) + (2 × -1.5)
           = 5 + 0 + (-3)
           = 2

건수 = 15 (중립 포함)
채널점수 = 2 / 15 = 0.133

커뮤니티 채널: 긍정 30건, 중립 20건, 부정 25건, 신고급 3건

가중감성합 = (30 × 0.5) + (20 × 0) + (25 × -0.7) + (3 × -1.5)
           = 15 + 0 + (-17.5) + (-4.5)
           = -7.0

건수 = 78
채널점수 = -7.0 / 78 = -0.090
```

### 4.3 결과값 범위 및 0~100 스케일

- **원시 SIR 범위**: -2.0 ~ +1.0
- **0~100 스케일**: `SIR_100 = ((SIR + 2) / 3) × 100`
  - 0 = 최악 (전부 신고대상급 부정)
  - 33.3 = 전부 일반 부정
  - 50 = 중립 (기준점)
  - 66.7 = 중립~긍정 사이
  - 100 = 전부 긍정

```typescript
const CHANNEL_WEIGHTS: Record<string, number> = {
  report: 5,
  news: 4,
  sns: 2,
  community: 2,
};

const SENTIMENT_SCORES: Record<string, Record<string, number>> = {
  report:    { positive: 1.0, neutral: 0, negative: -1.3, critical: -2.0 },
  news:      { positive: 1.0, neutral: 0, negative: -1.5, critical: -2.0 },
  sns:       { positive: 0.7, neutral: 0, negative: -0.8, critical: -1.5 },
  community: { positive: 0.5, neutral: 0, negative: -0.7, critical: -1.5 },
};

// 플랫폼 → 채널 매핑
const PLATFORM_CHANNEL: Record<string, string> = {
  naver_news: 'news',
  google_news: 'news',
  naver_blog: 'sns',
  youtube: 'sns',
  naver_cafe: 'community',
  stock_discussion: 'community',
};
```
